# Lexic Plugin Marketplace

This repository is a [Claude Code plugin marketplace](https://code.claude.com/docs/en/plugin-marketplaces) for [Lexic](https://lexic.io).

## Install

```
/plugin marketplace add lexic-io/Lexic.io-plugin
/plugin install lexic@lexic-marketplace
```

Then restart Claude Code. The first time you use a Lexic command, you'll be prompted to sign in.

## What's Included

The **lexic** plugin gives Claude Code persistent knowledge management, architectural decision tracking, and autonomous task orchestration.

See the [full plugin README](./plugins/lexic/README.md) for details on commands, skills, alternative client configs, and troubleshooting.
